# Kakao 소셜 로그인을 설정하세요

Kakao 커넥터는 애플리케이션이 Kakao의 OAuth 2.0 인증 시스템을 간결하게 사용할 수 있는 방법을 제공합니다.

<GuideTip />

## Kakao Developers Console 에서 프로젝트 설정하기 \{#set-up-a-project-in-the-kakao-devlopers-console}

- [Kakao Developers Console](https://developers.kakao.com/console/app) 을 방문하여 Kakao 계정으로 로그인하세요.
- **애플리케이션 추가**를 클릭하여 새 프로젝트를 생성하거나 기존 프로젝트를 선택하세요.

## Kakao 로그인 구성하기 \{#configure-kakao-login}

### Kakao 로그인 활성화하기 \{#activate-kakao-login}

- 메뉴에서 **제품 설정 -> Kakao 로그인**을 클릭하세요.
- `Kakao 로그인 활성화`를 켜세요.
- 아래 URL을 `Redirect URI`에 추가하세요.
  - `http(s)://YOUR_URL/callback/${connector_id}` (`connector_id`는 Logto Admin Console 커넥터 세부 정보 페이지의 상단 바에서 찾을 수 있습니다.)
  - (`YOUR_URL`을 `Logto` URL로 바꾸고, 상황에 맞게 `http` 또는 `https`를 선택하세요.)

### 개인정보 설정 \{#privacy-setting}

- 메뉴에서 **제품 설정 -> Kakao 로그인 -> 동의 항목**을 클릭하세요.
- `닉네임`, `프로필 이미지`, `이메일`의 상태를 **필수 동의**로 변경하세요. (프로젝트 설정에 따라 `이메일`을 **필수 동의**로 변경할 수 없을 수도 있습니다.)

### 보안 설정 (선택 사항) \{#security-setting-optional}

- 메뉴에서 **제품 설정 -> Kakao 로그인 -> 보안**을 클릭하세요.
- `클라이언트 비밀 코드`를 클릭하여 비밀 코드를 생성하세요.
- `활성화 상태`를 활성화로 변경하세요. (활성화하면 `비밀 코드`가 필요합니다.)

## Logto 구성하기 \{#configure-logto}

### 구성 유형 \{#config-types}

| 이름         | 유형    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId`는 프로젝트의 `REST API 키`입니다.
(Kakao developers console의 프로젝트 `요약`에서 찾을 수 있습니다.)

#### clientSecret \{#clientseceret}

`clientSecret`은 프로젝트의 `비밀 코드`입니다.
([보안 설정 (선택 사항)](#security-setting-optional)을 확인하세요.)
