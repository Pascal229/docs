우리는 또한 미리 정의된 스코프를 사용하는 데 도움이 되는 `LogtoUserScope` 열거형을 SDK 패키지에서 제공합니다.

```dart
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: [LogtoUserScope.email.value, LogtoUserScope.phone.value],
  // highlight-end
);
```
