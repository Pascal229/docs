import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

선호하는 패키지 관리자를 통해 Logto SDK 및 피어 종속성을 설치하세요:

<Tabs>

  <TabItem value="npm" label="npm">

```bash
npm i @logto/rn
npm i expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>
  <TabItem value="yarn" label="Yarn">

```bash
yarn add @logto/rn
yarn add expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add @logto/rn
pnpm add expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>

</Tabs>

`@logto/rn` 패키지는 Logto의 SDK입니다. 나머지 패키지들은 피어 종속성입니다. Expo CLI는 모든 네이티브 모듈의 종속성이 루트 프로젝트의 `package.json`에 직접 설치되어야 하기 때문에 직접 종속성으로 나열될 수 없습니다.

:::note
[베어 React Native 앱](https://docs.expo.dev/bare/overview)에 설치하는 경우, 이 [추가 설치 지침](https://docs.expo.dev/bare/installing-expo-modules/)도 따라야 합니다.
:::
